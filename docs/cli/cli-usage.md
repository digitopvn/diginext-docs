# CLI Usage

## Documentation

[CLI Commands & Subcommands](/docs/cli/cli-docs)

## Usage

```yaml
dx [commands] [subcommands] [--flags]
```

### Examples

```bash
- Start a Diginext server:
	$ export MONGODB_CONNECTION_STRING=...
	$ dx server up

- Login to Diginext server:
	$ dx login <workspace-dashboard-url>
```
