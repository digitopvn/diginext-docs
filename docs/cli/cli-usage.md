# CLI Usage

## Usage

```yaml
dx [commands] [subcommands] [--options]
```

### Examples

- Start a DXUP server:
	```bash
	$ export MONGODB_CONNECTION_STRING=...
	$ dx server up
	```

- Login to DXUP server:
	```bash
	$ dx login https://apps.diginext.site
	```
	```bash
	$ dx login <workspace-dashboard-url>
	```
